분위기 & 스타일

프로그램 전체의 "느낌"을 결정하는 키워드예요

모던 / 미니멀
Modern / Minimal
깔끔하고 현대적인 느낌. 꼭 필요한 것만 남긴 디자인.
  • 흰색·회색 위주의 무채색 팔레트
  • 여백(공백)을 넓게 활용
  • Apple, 노션이 대표적인 예
프롬프트 예시 "모던하고 미니멀한 UI, 불필요한 요소 제거, 흰색 배경"
귀여운
Cute / Kawaii
동글동글하고 밝은 색감. 아기자기한 느낌의 디자인.
  • 모서리가 둥근 버튼과 카드 (border-radius 크게)
  • 파스텔 핑크·민트·라벤더 색상
  • 이모지와 아이콘을 적극 활용
프롬프트 예시 "귀엽고 아기자기한 스타일, 둥근 버튼, 파스텔 핑크 색상"
레트로 / Y2K
Retro / Y2K
80~2000년대 인터넷 느낌. 반짝이고 알록달록한 스타일.
  • 네온 원색 (형광 핑크·노랑·하늘)
  • 픽셀·모노스페이스 폰트
  • 두꺼운 테두리, 별·하트 장식
프롬프트 예시 "Y2K 감성, 2000년대 초반 인터넷 홈페이지 느낌"
미래적 / 사이버펑크
Futuristic / Cyberpunk
SF처럼 첨단 기술 느낌. 네온·홀로그램·해킹 분위기.
  • 어두운 배경 + 형광 청록·분홍 텍스트
  • 모노스페이스(코드) 폰트
  • 격자 패턴, HUD 인터페이스 스타일
프롬프트 예시 "사이버펑크 스타일, 어두운 배경에 네온 청록색 텍스트"
유리 느낌
Glassmorphism
반투명 유리처럼 흐릿하고 투명한 레이어 효과.
  • 배경이 살짝 비치는 카드 (backdrop-filter: blur)
  • 그라데이션 배경 위에 올려야 효과적
  • 아이폰 잠금화면 같은 느낌
프롬프트 예시 "glassmorphism, 반투명 유리 카드, 그라데이션 배경"
입체적인
Neumorphism / 3D
실제 물건처럼 두께감과 그림자가 있는 디자인.
  • 밝고 어두운 그림자를 양쪽에 사용
  • 배경과 같은 색 계열의 카드
  • 버튼을 누르면 쏙 들어가는 느낌
프롬프트 예시 "neumorphism 스타일, 입체적인 버튼, 눌리는 효과"
스크랩북 / 콜라주
Scrapbook / Collage
사진·스티커·메모를 붙인 듯한 아날로그 감성 디자인.
  • 기울어진 사진, 테이프, 손글씨 느낌
  • 격자 배경지 + 노트 용지 질감
  • 다이어리·추억 앱·SNS 다이어리에 어울림
프롬프트 예시 "스크랩북 콜라주 스타일, 기울어진 포토카드, 마스킹테이프, 격자 배경"
팝아트
Pop Art
만화·광고처럼 원색과 굵은 선을 쓰는 강렬한 스타일.
  • 빨강·노랑·파랑 원색을 과감하게
  • 굵은 검정 외곽선 (두꺼운 border)
  • 이벤트·캠페인·홍보용 페이지에 어울림
프롬프트 예시 "팝아트 스타일, 원색 배경에 굵은 검정 테두리, 만화 말풍선"
자연스러운 / 코지
Natural / Cozy
자연에서 온 것 같은 따뜻하고 편안한 느낌.
  • 베이지·크림·갈색 계열 색상
  • 세리프(명조) 폰트로 따뜻한 느낌
  • 카페, 일기장, 독서 앱에 잘 어울림
프롬프트 예시 "코지하고 따뜻한 느낌, 베이지 색상, 카페 분위기"

색상 & 테마

어떤 색을 쓸지 결정하는 키워드예요

다크 모드
Dark Mode
배경이 어둡고 글자가 밝은 테마. 눈이 편하고 세련됨.
  • 밤에 봐도 눈이 덜 피로함
  • 개발자·게임 앱이 많이 사용
  • 네온 색상과 조합하면 멋짐
프롬프트 예시 "다크 모드, 검정 배경에 흰 텍스트, 어두운 카드"
파스텔 톤
Pastel
연하고 부드러운 색감. 보기에 포근하고 따뜻한 느낌.
  • 분홍·민트·라벤더·베이지 등 연한 색
  • 눈에 자극이 적고 부드러움
  • 다이어리·스터디·건강 앱에 적합
프롬프트 예시 "파스텔 색상, 연한 핑크와 민트와 라벤더 조합"
그라데이션
Gradient
두 가지 이상의 색이 자연스럽게 섞이는 효과.
  • 인스타그램 로고처럼 색이 자연스럽게 변함
  • 버튼·배경·카드에 활용
  • 생동감 있고 시각적으로 풍부함
프롬프트 예시 "보라→파랑→분홍 그라데이션 배경, 카드에도 그라데이션"
모노크롬
Monochrome
흑백 또는 단색 계열만 사용한 절제된 색감.
  • 검정·흰색·회색만으로 세련미 표현
  • 하나의 색 톤을 단계별로 쓰는 방법도 있음
  • 사진 포트폴리오, 패션·아트 앱에 어울림
프롬프트 예시 "모노크롬, 흑백 팔레트, 회색 계열만 사용한 미니멀 UI"
네온 색상
Neon
형광빛의 강렬한 색. 게임·파티·클럽 분위기.
  • 어두운 배경일 때 더 빛남
  • 형광 초록·분홍·파랑·보라 사용
  • 에너지 넘치는 에너지 표현에 최적
프롬프트 예시 "검정 배경에 네온 초록·분홍·파랑 컬러 사용"
🔴

레이아웃 & 구조

화면을 어떻게 배치할지 결정하는 키워드예요

채팅 / 메신저형
Chat / Messenger
카카오톡·문자처럼 말풍선으로 대화하는 UI.
  • 내 메시지는 오른쪽, 상대방은 왼쪽
  • AI 챗봇·상담 앱·Q&A 앱에 딱 맞음
  • 자연스럽고 친근한 인터랙션 흐름
프롬프트 예시 "채팅 UI, 말풍선 레이아웃, 오른쪽 내 메시지·왼쪽 AI 답변"
타임라인형
Timeline
시간 순서대로 사건·기록을 세로로 나열하는 구조.
  • 할 일·일정·히스토리 관리 앱에 적합
  • 완료·진행중·예정을 색으로 구분
  • 인스타그램 스토리 같은 시간 흐름 표현
프롬프트 예시 "타임라인 UI, 세로 시간축, 완료/진행/예정 상태 색상 구분"
대시보드형
Dashboard
여러 정보와 차트를 한 화면에 모아 보여주는 구조.
  • 숫자·그래프·통계를 한눈에
  • 왼쪽 사이드바 + 오른쪽 콘텐츠 구조
  • 공부 현황, 용돈 관리 앱에 적합
프롬프트 예시 "대시보드 형식, 왼쪽 사이드바 + 통계 카드 + 차트"
카드형 레이아웃
Card Layout
정보를 카드 모양의 박스에 담아 그리드로 정렬.
  • 트위터·에어비앤비 같은 박스 단위 정보
  • 스크롤하며 보기 편함
  • 메모·기록·일정 앱에 잘 어울림
프롬프트 예시 "카드형 레이아웃, 3열 그리드, 각 카드에 이모지와 제목"
한 페이지 스크롤
Single Page Scroll
페이지 이동 없이 아래로 스크롤하며 보는 구조.
  • 소개 페이지·포트폴리오에 많이 사용
  • 스토리를 순서대로 이어서 보여주기 좋음
  • 모바일에서 특히 사용하기 편함
프롬프트 예시 "한 페이지 스크롤, 섹션별 풀스크린, 스크롤 내비게이션"

효과 & 애니메이션

화면이 움직이거나 전환될 때 쓰는 효과예요

스켈레톤 로딩
Skeleton Loading
데이터가 불러와지는 동안 회색 뼈대를 먼저 보여주는 효과.
  • 빈 화면 대신 레이아웃 모양을 미리 표시
  • 유튜브·인스타그램이 사용하는 방식
  • 기다리는 느낌을 줄여주는 UX 기법
프롬프트 예시 "스켈레톤 로딩 UI, 데이터 로딩 중 회색 플레이스홀더 표시"
스와이프 카드
Swipe Card
카드를 좌우로 밀어서 선택·거절하는 인터랙션.
  • 틴더·데이팅앱이 대표적인 예
  • 퀴즈·플래시카드 앱에도 활용 가능
  • 스택처럼 쌓인 카드 여러 장 구조
프롬프트 예시 "스와이프 카드 UI, 좌우로 밀어 선택, 스택 카드 효과"
진행바
Progress Bar
작업 진행 상황을 시각적으로 보여주는 막대.
  • "오늘 목표 75% 달성!" 같은 표현
  • 색상과 %로 성취감을 시각화
  • 습관·운동·공부 앱의 핵심 요소
프롬프트 예시 "목표 달성률을 컬러 진행바와 퍼센트로 표시"
호버 효과
Hover Effect
마우스를 올리면 색이나 크기가 바뀌는 효과.
  • 버튼·카드에 마우스 올리면 색 변경
  • "클릭할 수 있어요"라는 힌트를 시각적으로 줌
  • 인터랙티브하고 살아있는 느낌
프롬프트 예시 "모든 버튼과 카드에 호버 시 색상 변화 효과 추가"
파티클 / 반짝 효과
Particle / Sparkle Effect
화면에 작은 점들이 흩날리는 배경 효과.
  • 목표 달성·레벨업 시 폭죽 효과
  • 배경에 별이 떠다니는 우주 분위기
  • 게임·이벤트 화면에 특히 효과적
프롬프트 예시 "목표 달성 시 파티클 효과, 배경에 별 떠다니는 애니메이션"

느낌 & 감성

앱의 "분위기"를 완성하는 마지막 키워드예요

미니멀리스트
Minimalist
화면에 딱 하나의 핵심만 남기는 극도로 단순한 디자인.
  • "모던/미니멀"보다 더 극단적으로 비움
  • 버튼도 1개, 텍스트도 1줄
  • 명상·집중 앱, 타이머 앱에 어울림
프롬프트 예시 "극도로 미니멀, 단 하나의 액션만 화면에, 흰 배경 여백 가득"
게임 느낌
Gamified
점수·레벨·배지 등 게임 요소를 녹인 디자인.
  • 공부·운동을 게임처럼 → 더 재미있어짐
  • "레벨업!", "연속 달성 🔥" 피드백
  • 두올링고·포켓몬GO가 대표적
프롬프트 예시 "게이미피케이션, 레벨·XP·배지 시스템, 퀘스트 목록"
프로페셔널
Professional
업무용처럼 정돈되고 신뢰감 있는 디자인.
  • 네이비·회색·흰색 계열 색상
  • 정돈된 표와 데이터 중심 표현
  • 발표 자료·포트폴리오·업무 앱에 적합
프롬프트 예시 "프로페셔널한 기업 스타일, 네이비·흰색, 데이터 중심"
아티스틱
Artistic
그림이나 예술 작품 같은 개성 있는 디자인.
  • 손 그림, 수채화, 콜라주 스타일
  • 정해진 규칙 없이 자유로운 레이아웃
  • 창작·갤러리·포트폴리오 앱에 어울림
프롬프트 예시 "아티스틱, 손 그림 느낌, 불규칙한 도형과 수채화 색감"

타이포그래피 & 폰트

글자 스타일로 디자인의 성격을 완성해요

굵은 타이틀형
Bold Headline
아주 큰 글자 하나가 화면을 꽉 채우는 임팩트 있는 스타일.
  • 제목을 화면의 절반 이상으로 키움
  • 잡지·포스터 같은 강렬한 인상
  • 소개 페이지·이벤트 랜딩 페이지에 적합
프롬프트 예시 "큰 굵은 헤드라인 폰트, 화면을 꽉 채우는 타이틀, 잡지 스타일"
손글씨 느낌
Handwriting Style
직접 쓴 것 같은 필기체·손글씨 느낌의 폰트 스타일.
  • 노트 줄·여백선과 함께 쓰면 찰떡
  • 일기장·편지·메모 앱 분위기에 딱
  • 친근하고 개인적인 감성을 표현
프롬프트 예시 "손글씨 폰트, 노트 줄 배경, 필기체 스타일 UI"
코드 / 터미널 폰트
Code / Monospace
코딩 에디터·터미널처럼 고정폭 폰트를 사용하는 스타일.
  • 글자 너비가 모두 같은 모노스페이스 폰트
  • 개발·해킹·SF 느낌을 자연스럽게 연출
  • 코드 에디터, 터미널 테마 앱에 잘 어울림
프롬프트 예시 "모노스페이스 폰트, 터미널 UI, 신택스 하이라이팅 색상"
에디토리얼 / 잡지형
Editorial
신문·잡지처럼 텍스트 컬럼과 타이포를 핵심으로 쓰는 스타일.
